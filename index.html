Memory

Projet de DUT

Download as .zip Download as .tar.gz View on GitHub

Contexte

Projet réalisé durant la deuxième année de DUT Informatique.

Sujet

Réalisation d'un jeu Memory devant comporter trois niveaux de difficulté. Le thème était libre.

Technologie

La gestion du jeu est uniquement basé sur le Javascript. L'AJAX est utilisé pour la gestion des scores.

Installation

  1. Télécharger le dossier.
  2. Extraire l'archive dans votre serveur local.
  3. Créer et importer la base de données. Score.sql

  4. Modifier le fichier Score.php situé à la racine :

    ligne 2 : $cnct = mysqli_connect("host","username" ,"password", "database");
